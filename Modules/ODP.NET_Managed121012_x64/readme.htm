This document provides information regarding the contents on the zip file and the steps required to setup ODP.NET, Managed Driver xcopy installation.

General Information

  1. The files of this zip file is NOT to be installed on top of an existing Oracle Universal Installer (OUI) based Oracle Home installation.

  2. After unzipping the files, the product must be configured by executing the approporiate configure.bat.

Installation Instructions

  1. Download the ODP.NET, Managed Driver (xcopy version) zip file to a directory for staging.
  2. Unzip ODP.NET, Managed Driver (xcopy version) zip file to expand the contents to your target install location.

    The following directories will be created:

    a) odp.net - ODP.NET, Managed Driver files
    b) network - sample .ORA files (i.e. tnsnames.ora and sqlnet.ora)

  3. Execute the appropriate configure.bat to complete ODP.NET, Managed Driver setup. The configure.bat files are located in the following directories:

    • For 32-bit .NET Framework: .\odp.net\managed\x86
    • For 64-bit .NET Framework: .\odp.net\managed\x64

    ODP.NET, Managed Driver already includes distributed transaction enlistment and commit services natively for distributed transactions. The OraMTS README contains additional information on distributed transaciton recovery service setup with ODP.NET, Managed Driver.

    Read the "Oracle Data Provider for .NET Developer's Guide" documentation for more information on setup and use of ODP.NET, Managed Driver.

    Each directory contains an unconfigure.bat if ODP.NET, Managed Driver needs to be unconfigured from the machine.
